Oh boy, this is a wild one! Found Daniel A. Books in the 1870 census in Greene County, TN. Then in 1880, suddenly there's only one Jacob Chambers where there used to be two... 🤔

The Tennessee State Library and Archives has some interesting court records from 1879 about the "original" Jacob Chambers' property being transferred to "Jacob E. Chambers" - very unusual for that time.
