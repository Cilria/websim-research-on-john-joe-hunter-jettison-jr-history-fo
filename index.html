Rubidium was a proprietary web browser developed by Rubidium Software S.A., a French software company based in Lyon. First released in 1999, Rubidium was marketed as "the fastest and most powerful web browser" of its era, targeting professional users and power users on Windows 98 and later Microsoft Windows operating systems.

History

Rubidium Software S.A. was founded in 1998 by former INRIA researchers Jean-Paul Mercier and Sophie Laurent. The company initially focused on developing high-performance software for Windows 98, recognizing the emerging need for advanced web browsing capabilities as the internet became mainstream.

The browser's development began in late 1998, with the first public beta released in May 1999. The official 1.0 release launched on June 15, 1999, at the Internet World Paris conference. Rubidium gained initial attention for its speed optimizations and advanced features not found in contemporary browsers like Internet Explorer 5.0 and Netscape Navigator 4.5.

Features and Technology

Rubidium was built using a custom rendering engine called RRender, developed specifically for high-performance HTML parsing and rendering. Key features included:

  • Tabbed browsing with detachable tabs
  • Built-in download manager with resume capability
  • Advanced JavaScript engine with JIT compilation
  • Integrated development tools including DOM inspector
  • Support for CSS 2.1 and early CSS 3 features
  • Customizable interface with XML-based skinning system
  • Built-in ad blocking and pop-up suppression
  • Encrypted bookmark synchronization

Reception and Market Performance

Upon release, Rubidium received positive reviews from technology publications. PC Expert magazine praised its "blazing speed" and "innovative features," awarding it 4.5 out of 5 stars. Le Monde Informatique highlighted its "professional-grade tools" and "French engineering excellence."

Despite strong technical reviews, Rubidium struggled to gain significant market share against Internet Explorer's dominance and the rising popularity of Mozilla Firefox. By 2001, the browser had approximately 2.3 million active users, primarily in France, Germany, and Japan.

Later Development and Decline

Version 2.0, released in October 2000, introduced significant performance improvements and support for emerging web standards. However, development slowed as the company faced financial difficulties during the dot-com bust.

The final major release, version 3.0, launched in September 2002, adding Windows XP compatibility and enhanced security features. Despite these updates, user adoption continued to decline as Internet Explorer 6.0 and Firefox gained market dominance.

Legacy

Rubidium Software S.A. ceased operations in March 2003 due to bankruptcy. The browser's website was taken offline in June 2003, and official support ended. Several former Rubidium engineers later contributed to the development of alternative browsers, including early versions of Opera and Safari for Windows.

Rubidium is remembered among web historians as an innovative browser that introduced several features that would become standard years later, including tabbed browsing, integrated development tools, and built-in ad blocking. The browser's source code was never released, but several enthusiasts have created unofficial archives and compatibility patches for modern Windows systems.
