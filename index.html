Remember Rubidium? This thing was WAY ahead of its time

Just stumbled across an old installer CD for Rubidium 2.0 while cleaning out my dad's basement. Holy shit, I forgot how absolutely insane this browser was for 1999. Tabbed browsing before Firefox existed, built-in ad blocking, a goddamn DOM inspector when most people were still using IE5. It even had encrypted bookmark sync!

Shame it died in the dot-com crash. Anyone else remember this beast? Apparently it peaked at like 2.3 million users which is wild considering how niche it was.

browserHistorian • 298 points • 30 minutes ago
Fun fact: Rubidium was actually the first browser to implement what we'd now call "private browsing." They called it "Stealth Mode" and it was buried like 3 menus deep. Also had this crazy feature where you could right-click any element and "freeze" the page state - basically taking a snapshot of the entire DOM. Web devs used it for debugging before we had proper dev tools.
